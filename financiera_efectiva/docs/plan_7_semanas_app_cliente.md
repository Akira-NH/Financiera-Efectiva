# Plan de 7 semanas - App Cliente

## Semana 1 - Base del proyecto

- Proyecto Flutter base.
- Branding institucional.
- Splash screen.
- Navegacion principal.
- Estructura de carpetas limpia.
- Pantallas navegables iniciales.

## Semana 2 - Autenticacion y usuarios

- Login funcional.
- Registro de cliente.
- Validaciones.
- Persistencia local de sesión.

## Semana 3 - Dashboard principal

- Saldo general.
- Últimos movimientos.
- Tarjetas de resumen visual.

## Semana 4 - Modulos financieros centrales

- Ahorros: saldo, depositos y estados de cuenta.
- Créditos: préstamos activos, cuotas pendientes y cronograma.

## Semana 5 - Operaciones y formularios

- Transferencias.
- Pagos.
- Confirmación de operación.
- Historial.

## Semana 6 - Perfil y configuracion

- Perfil editable.
- Configuración.
- Seguridad basica.

## Semana 7 - Cierre, QA y presentacion

- Correccion de bugs.
- Mejoras UI/UX.
- APK final.
- Manual corto.
